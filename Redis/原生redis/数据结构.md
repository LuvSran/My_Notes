### 1. String（字符串）

- **设置/获取**：`SET key value` / `GET key`
    
- **仅不存在时设置**：`SET key value NX`（常用作分布式锁）
    
- **设置过期时间**：`SETEX key seconds value`
    
- **自增/自减**：`INCR key` / `DECR key` / `INCRBY key increment`
    
- **批量操作**：`MSET k1 v1 k2 v2` / `MGET k1 k2`
    

### 2. List（列表）

- **两端插入**：`LPUSH key value`（左） / `RPUSH key value`（右）
    
- **两端弹出**：`LPOP key` / `RPOP key`
    
- **范围查询**：`LRANGE key start stop`（`0 -1` 表示获取全部）
    
- **长度**：`LLEN key`
    
- **阻塞弹出**：`BLPOP key timeout`（队列为空时等待，常用于消息队列）
    

### 3. Hash（哈希）

- **设置/获取字段**：`HSET key field value` / `HGET key field`
    
- **批量设置**：`HMSET key f1 v1 f2 v2`
    
- **获取所有键值**：`HGETALL key`
    
- **获取所有字段名/所有值**：`HKEYS key` / `HVALS key`
    
- **字段自增**：`HINCRBY key field increment`
    

### 4. Set（集合）

- **添加/移除**：`SADD key member` / `SREM key member`
    
- **判断成员是否存在**：`SISMEMBER key member`
    
- **获取所有成员**：`SMEMBERS key`
    
- **随机获取**：`SRANDMEMBER key [count]` / `SPOP key`（随机弹出）
    
- **集合运算**：`SINTER k1 k2`（交集） / `SUNION k1 k2`（并集） / `SDIFF k1 k2`（差集）
    

### 5. ZSet（有序集合）

- **添加/更新**：`ZADD key score member`
    
- **获取分数**：`ZSCORE key member`
    
- **范围获取（升序）**：`ZRANGE key start stop [WITHSCORES]`
    
- **范围获取（降序）**：`ZREVRANGE key start stop [WITHSCORES]`
    
- **按分数段统计**：`ZCOUNT key min max`
    
- **查看排名**：`ZRANK key member`（从小到大排名，0开始）
