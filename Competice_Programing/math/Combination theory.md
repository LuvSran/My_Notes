# 曲棍球公式
$对于整数 ( n \geq r \geq 0 )$，有：  $$\sum_{k=r}^{n} \binom{k}{r} = \binom{n+1}{r+1}$$
# Vandermonde 恒等式
$$\sum_{i=0}^{k} \binom{m}{i}\binom{n}{k-i} = \binom{n+m}{k}$$

# 隔板法
将 n个​​相同的物品​​分配到 m个​​不同的位置​​(每个位置可放0个或多个物品），方案数为：
$$\binom{n+m-1}{n} 或 \binom{n+m-1}{m-1}$$
