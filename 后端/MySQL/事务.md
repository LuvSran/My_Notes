```sql
#事务简介
-- 事务是一组操作的集合，要么全部成功，要么全部失败

#事务操作
START TRANSACTION; -- 开启事务
COMMIT;            -- 提交事务（永久保存）
ROLLBACK;          -- 回滚事务（撤销所有变更）

#事务四大特性（ACID）
-- 原子性(Atomicity)：不可分割，全成功或全失败
-- 一致性(Consistency)：执行前后数据状态合法
-- 隔离性(Isolation)：并发事务互不干扰
-- 持久性(Durability)：提交后数据永久保存

#并发事务问题
-- 脏读：读到未提交的数据
-- 不可重复读：同一事务内多次读取结果不同
-- 幻读：读取范围变化（新增/删除行）

#事务隔离级别
READ UNCOMMITTED;   -- 允许脏读
READ COMMITTED;     -- 防止脏读
REPEATABLE READ;    -- 防止脏读、不可重复读（MySQL默认）
SERIALIZABLE;       -- 最高隔离，防止所有并发问题
```
