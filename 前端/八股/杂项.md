# 讲讲SEO
SEO 中文叫 **搜索引擎优化**。
简单来说，SEO 就是通过技术和内容手段，让你的网站在百度、谷歌等搜索引擎的“自然搜索结果”中排名更靠前，从而获得更多免费流量。

| 维度 | 核心任务 | 关键动作 |
| ---- | ---- | ---- |
| **技术 SEO**  <br>(Technical SEO) | 让搜索引擎**找得到、读得懂** | 提升网站加载速度（<3秒）、适配移动端（手机流量占比超60%）、使用 HTTPS 安全协议、提交站点地图 (Sitemap)。 |
| **内容 SEO**  <br>(On-Page SEO) | 让搜索引擎觉得**有价值** | 挖掘用户搜索的**关键词**（尤其是长尾词）、撰写高质量原创内容、优化标题 (Title) 和描述 (Description)、图片添加 Alt 标签。 |
| **站外 SEO**  <br>(Off-Page SEO) | 让搜索引擎觉得**有权威** | 获取其他高质量网站的链接（**外链**）。这相当于别的网站给你投了“信任票”，外链越多越权威，排名越稳。 |
