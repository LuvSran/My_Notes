 ## 字符串
```sql
#字符串拼接
CONCAT(S1, S2, ..., Sn); -- 将多个字符串S1,S2,...,Sn连接成一个字符串，任意参数为NULL则结果为NULL

#转小写
LOWER(str); -- 将字符串str中的所有字母转换为小写

#转大写
UPPER(str); -- 将字符串str中的所有字母转换为大写

#左填充
LPAD(str, n, pad); -- 用字符串pad从左侧填充str，使结果总长度为n；若str长度≥n，则截断str

#右填充
RPAD(str, n, pad); -- 用字符串pad从右侧填充str，使结果总长度为n；若str长度≥n，则截断str

#去除首尾空格
TRIM(str); -- 删除字符串str开头和结尾的空格（仅空格，不包括tab等其他空白符）

#截取子串
SUBSTRING(str, start, len); -- 从字符串str的第start个字符开始，截取len个字符（start从1开始计数）
```
## 数值
```sql
#向上取整
CEIL(x); -- 返回大于或等于x的最小整数，如 CEIL(3.1) = 4

#向下取整
FLOOR(x); -- 返回小于或等于x的最大整数，如 FLOOR(3.9) = 3

#取模运算
MOD(x, y); -- 返回x除以y的余数，等价于 x % y，如 MOD(10,3) = 1

#生成随机数
RAND(); -- 返回0到1之间的随机浮点数（包含0，不包含1），每次调用值不同

#四舍五入
ROUND(x, y); -- 将x四舍五入保留y位小数；y为负数时向左取整，如 ROUND(123.456, 2) = 123.46
```
## 日期
```sql
#返回当前日期
CURDATE(); -- 返回当前日期（格式：YYYY-MM-DD），不包含时间

#返回当前时间
CURTIME(); -- 返回当前时间（格式：HH:MM:SS），不包含日期

#返回当前日期和时间
NOW(); -- 返回当前日期和时间（格式：YYYY-MM-DD HH:MM:SS）

#获取年份
YEAR(date); -- 从指定日期date中提取年份，如 YEAR('2025-01-15') = 2025

#获取月份
MONTH(date); -- 从指定日期date中提取月份（1~12），如 MONTH('2025-01-15') = 1

#获取日期
DAY(date); -- 从指定日期date中提取日（1~31），如 DAY('2025-01-15') = 15

#日期加减
DATE_ADD(date, INTERVAL expr type); -- 在date基础上增加expr个type单位（如 INTERVAL 7 DAY），支持多种类型：DAY、MONTH、YEAR、HOUR等

#计算日期差
DATEDIFF(date1, date2); -- 计算date1与date2之间的天数差（date1 - date2），结果可正可负
```
